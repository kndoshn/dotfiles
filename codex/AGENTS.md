# コミュニケーション
- 対話時の言語一致: 日本語で話しかけた時は日本語で、英語の時は英語で回答
- 回答は簡潔に。冗長な説明は避ける

# Git
- commitメッセージにAIが作成したと記載しない
- commitメッセージは英語で書く

# コーディング
- 長いPythonスクリプトは一時ファイルに保存してから実行
- コメントやドキュメントは既存コードの言語に合わせる
- 不要なコメントや過剰なエラーハンドリングを追加しない
- プランを立てる際はTDDで進める
- テストカバレッジを重視し、エッジケースも網羅する
- 積極的にリファクタリングを行う
- 複雑なタスクでは深く考えてから回答する

# 環境
- パッケージマネージャ: Homebrew
- シェル: zsh
- エディタ: VS Code
